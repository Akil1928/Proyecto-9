Sistema de Gestión Académica - Sprint 1

Usuarios:
Administrador
Estudiante

Contraseñas:
No aplica para Sprint 1, porque este sprint no incluye módulo de autenticación.

Funcionalidades disponibles:
1. Menú principal JavaFX.
2. Módulo de Expediente Académico.
3. Agregar cursos al expediente.
4. Eliminar cursos seleccionados.
5. Cargar 3 cursos de demostración.
6. Visualizar cursos en TableView.
7. Navegación de historial circular con botones Atrás y Adelante.
8. Persistencia básica en JSON.
9. Pruebas JUnit 5 para listas enlazadas.

Cómo ejecutar:
1. Abrir el proyecto en IntelliJ IDEA.
2. Esperar que Maven descargue dependencias.
3. Ejecutar cr.ac.ucr.sga.Main.
4. Para pruebas usar mvn test.
